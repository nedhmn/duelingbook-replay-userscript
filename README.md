# Duelingbook Replay UserScripts

Collection of userscripts for duelingbook replays.

## Installation (Chrome)

1. Install [tampermonkey chrome extension](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en&pli=1)
2. Click on this `create new script` button in the tampermonkey dashboard:

<img width="1270" height="474" alt="image" src="https://github.com/user-attachments/assets/d240585d-71a3-465b-a6ba-e30241570fd0" />

3. Once your in the script editor, replace all of the contents of the editor with a [userscript](./userscripts).
4. Press `Ctrl + s` (if on Windows) to save the userscript.
5. Go to a duelingbook replay to try out the userscript.
